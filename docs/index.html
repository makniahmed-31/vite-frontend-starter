vite-frontend-starter v1.0.0

Vite Frontend Starter is a boilerplate kit for easy building modern static web-sites using Vite builder.

How to use

Instalation

  1. git clone https://github.com/zakandaiev/vite-frontend-starter.git - clone or download repository from GitHub
  2. cd vite-frontend-starter - go to the folder
  3. npm i - install packages

Development

  • npm run dev - start development mode with live-server

Building

  • npm run build or npm run prod - build static files for production
  • npm run preview - start server for build preview

Features

  • Modern Vite environment for development
  • Twig templating engine
  • Well thought-out and convenient project structure
  • HTML5 and CSS3 ready
  • SEO friendly
  • SASS/SCSS preprocessor
  • Autoprefixer
  • Live-server with hot-reload
  • HTML, CSS, JS, images auto minifier
  • Ready-to-use Javascript utils, HTML styled components, CSS helpers, SCSS utils etc.
  • reseter.css
  • .htaccess, robots.txt, sitemap.xml, favicon
  • 404 page
  • And many more...
